# CMS-style-blog-site

## Purpose 
The purpose of this project was to create a CMS style blog site so users who want to write about tech can publish articles, blog posts, and thoughts and opinions.


## Dependencies
•Bcrypt <br>
•Sequelize <br>
•Express <br>
•dotenv <br>
•Handlebars <br>
•Express-session <br>
•Mysql2 <br>
•Jest <br>

Powered by Node.
